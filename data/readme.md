Данная директория предназначена для хранения исходного датасета DEAM, промежуточных обработанных файлов и извлечённых аудиопризнаков. Данные не включены в репозиторий из-за объёма файлов и лицензионных ограничений; для воспроизведения экспериментов необходимо разместить датасет локально в папке data/raw/DEAM/.
