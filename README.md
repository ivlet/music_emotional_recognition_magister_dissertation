# Music Emotion Recognition на основе спектральных признаков аудиосигнала

Репозиторий содержит исследовательский ML-пайплайн для задачи **классификации эмоциональной окраски музыкальных треков** на основе аудиопризнаков и моделей машинного обучения.

Проект выполнен в рамках магистерской выпускной квалификационной работы на тему:

> **«Методы классификации эмоциональной окраски музыкальных треков на основе спектральных признаков аудиосигнала»**

Основная идея работы — исследовать, насколько эффективно можно классифицировать эмоциональное состояние музыкального трека по признакам, извлечённым из аудиосигнала. В проекте рассматриваются как классические модели машинного обучения, так и нейросетевые подходы, включая последовательностные модели и предобученные аудиотрансформеры.

---

## Цель проекта

Цель проекта — разработать и сравнить методы классификации эмоциональной окраски музыкальных треков на основе спектральных и связанных с ними аудиопризнаков.

В рамках проекта решаются следующие задачи:

- изучение предметной области Music Emotion Recognition;
- анализ корпуса DEAM;
- формирование статической и динамической постановки задачи;
- извлечение аудиопризнаков из музыкальных треков;
- построение признаковых представлений для классических моделей и нейронных сетей;
- обучение и сравнение нескольких групп моделей;
- анализ качества классификации и типичных ошибок.

---

## Используемый датасет

В проекте используется датасет **DEAM** — Database for Emotional Analysis in Music.

Датасет содержит музыкальные треки и аннотации эмоционального состояния в терминах двух измерений:

- **valence** — эмоциональная валентность, условно отражающая позитивность/негативность восприятия;
- **arousal** — уровень эмоционального возбуждения или интенсивности.

На основе этих двух измерений формируется классификация по эмоциональным квадрантам модели Russell:

| Квадрант | Valence | Arousal | Интерпретация |
|---|---:|---:|---|
| Q1 | высокая | высокая | радостное / энергичное состояние |
| Q2 | низкая | высокая | напряжённое / тревожное состояние |
| Q3 | низкая | низкая | грустное / спокойное состояние |
| Q4 | высокая | низкая | расслабленное / умиротворённое состояние |

## Постановки задачи

В проекте рассматриваются две основные постановки.

### 1. Статическая классификация

В статической постановке каждому музыкальному треку сопоставляется один итоговый эмоциональный класс.

Для этого используются агрегированные значения аннотаций valence/arousal и признаки, извлечённые из аудиосигнала целиком.

Пример признакового представления:

```text
track_id → [MFCC, chroma, spectral centroid, spectral bandwidth, RMS, ZCR, tempo, ...] → emotion_class
```

### 2. Динамическая классификация

В динамической постановке эмоциональное состояние рассматривается во времени.

Аудиотрек делится на окна фиксированной длины, для каждого окна извлекаются признаки, после чего модель обучается учитывать временную структуру сигнала.

Пример:

```text
track_id → последовательность окон → последовательность признаков → emotion_class
```

## Используемые признаки

В проекте используются аудиопризнаки, отражающие спектральные, тембральные и энергетические характеристики сигнала:

- **MFCC** — мел-кепстральные коэффициенты;
- **Chroma features** — распределение энергии по музыкальным высотным классам;
- **Spectral centroid** — условный «центр масс» спектра;
- **Spectral bandwidth** — ширина спектрального распределения;
- **Spectral rolloff** — частотная граница, ниже которой сосредоточена основная энергия сигнала;
- **Zero-crossing rate** — частота пересечений сигнала с нулевым уровнем;
- **RMS energy** — среднеквадратичная энергия сигнала;
- **Tempo** — оценка темпа музыкального трека.

Для классических моделей признаки агрегируются с помощью статистик:

- среднее значение;
- стандартное отклонение;
- минимум;
- максимум;
- медиана.

## Используемые модели

В проекте сравниваются несколько групп моделей.

### Классические модели машинного обучения

- Logistic Regression;
- K-Nearest Neighbors;
- Support Vector Machine;
- Random Forest;
- MLP-классификатор.

### Нейросетевые последовательностные модели

- LSTM;
- GRU;
- CRNN.

### Предобученные аудиомодели

Также исследуется использование предобученных моделей для аудио, включая трансформерные архитектуры и модели, обученные на больших аудиокорпусах.

Такие модели позволяют получать более выразительные представления аудиосигнала по сравнению с ручным набором признаков.

## Метрики качества

Для оценки качества классификации используются:

- **accuracy**;
- **precision**;
- **recall**;
- **F1-score**;
- **macro F1-score**;
- confusion matrix.

## Научный контекст

Music Emotion Recognition — это направление на стыке обработки аудиосигналов, музыкальной информатики и машинного обучения.

Задача является сложной по нескольким причинам:

- эмоции в музыке субъективны;
- разные слушатели могут по-разному воспринимать один и тот же трек;
- эмоциональное состояние может меняться внутри композиции;
- музыкальный сигнал содержит большое количество акустических характеристик;
- небольшие датасеты ограничивают возможности обучения глубоких моделей с нуля.

В связи с этим в работе сравниваются несколько подходов: от классических моделей на заранее извлечённых признаках до нейросетевых архитектур и предобученных аудиомоделей.
